# Dot Art Editer

ブラウザで動作するドット絵（ピクセルアート）エディタです。  
TypeScript + Vite で構築されており、Canvas 2D API のみで描画しています。

## 機能

### ツール

| ツール | 説明 |
|--------|------|
| ペン | 現在の色でピクセルを塗る |
| 消しゴム | ピクセルを透過（黒）に戻す |
| スポイト | クリックしたピクセルの色を取得する |
| 選択 | ドラッグで矩形範囲を選択する |
| ▼ 暗く | クリックしたピクセルの色を暗くする（×0.9） |
| ▲ 明るく | クリックしたピクセルの色を明るくする（×1.1） |
| スムース | 周囲8ピクセルとブレンドして色を滑らかにする |

### 選択操作（範囲選択後に使用）

| 操作 | ショートカット |
|------|----------------|
| コピー | `Ctrl+C` |
| 切り取り | `Ctrl+X` |
| 移動モード開始 | `Ctrl+M` |
| 左右反転 | `Ctrl+H` |
| 貼り付け | `Ctrl+V`（クリックで配置、`Esc` で解除） |

### 表示・ナビゲーション

- **表示範囲**: 横・縦のドット数を指定（デフォルト 128×48）
- **ズーム**: 1ドットを画面上の何 px で表示するか調整（1〜40）
- **ページ送り**: ↑↓←→ ボタンで表示範囲ぶん移動
- **グリッド線**: 表示/非表示の切り替え
- **全体マップ（ミニマップ）**: ドキュメント全体を縮小表示。クリックで表示位置を移動

### ガイド画像（テンプレート）

参照用の画像をドキュメント上に半透明で重ねて表示できます。  
不透明度はスライダーで調整可能です。

### アニメーションプレビュー

現在の表示範囲を水平方向に等分割してフレームとして扱い、ループ再生します。  
フレーム数（1〜16）・FPS（1〜60）・ズーム倍率（1〜8）を調整できます。

### ファイル操作

- **PNG で保存**: ドキュメントを PNG としてダウンロード（黒 = 透過）
- **PNG を読み込み**: ローカルの PNG をドキュメントへ取り込む（透過 → 黒に変換）
- **サンプルから読み込み**: `site-config.ts` で指定した URL の PNG を読み込む
- **サンプルをテンプレートに**: 同 URL の画像をガイド画像として読み込む
- **全消去**: ドキュメントをクリア

## ドキュメント仕様

| 項目 | 値 |
|------|----|
| サイズ | 1024 × 1024 px |
| 透過表現 | RGB (0, 0, 0) を透過として扱う |
| カラーモード | RGB（アルファは内部では不透明で固定） |

## 技術スタック

- **TypeScript** — 型付き実装
- **Vite 5** — 開発サーバー・バンドラー
- **Canvas 2D API** — 描画（外部描画ライブラリなし）

## 開発

### 必要環境

- [Deno](https://deno.com/) v2 以降、または Node.js

### セットアップ

```bash
# Deno
deno install
deno task dev

# Node.js / npm
npm install
npm run dev
```

### ビルド

```bash
# Deno
deno task build

# npm
npm run build
```

ビルド成果物は `dist/` に出力されます。

## サイト設定

`src/site-config.ts` を編集することで公開環境向けの設定を変更できます。

```ts
export const siteConfig = {
  // images/ ディレクトリのライブラリ表示（Web 公開時は false）
  showImageLibrary: false,

  // サンプル画像の URL（'' でボタン非表示）
  sampleImageUrl: '/sample.png',
};
```
